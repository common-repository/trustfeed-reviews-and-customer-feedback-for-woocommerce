== Installation ==

**Minimum requirements**

* WordPress 3.5.1 or greater
* PHP version 5.2.0 or greater
* WooCommerce 3.4.5 or greater

With your WooCommerce shop up and running, it’s time to install TrustFeed Reviews and Customer Feedback for WooCommerce. If you have an existing WooCommerce site and want to install TrustFeed, using the WordPress Admin is the most straightforward option as it handles everything for you. 

= A) Install and activate plugin =

1. Go to your website
2. Go to: Dashboard > Plugins > Add New > Search Plugins
3. Search for ‘TrustFeed Rating and Reviews’.
4. Select Install Now when you see it’s by TrustFeed
5. Select Activate Plugin


= B) Get API key =

1. Create a free account at [TrustFeed](https://admin.trustfeed.co/ "TrustFeed")
2. Go to [TrustFeed](https://admin.trustfeed.co/ "TrustFeed") Dashboard > Settings > Copy your API key
3. Go to Wordpress > TrustFeed> Settings
4. Copy API key in WordPress dashboard > TrustFeed>Settings
5. Click save changes
6. You’re ready to start collecting feedback
